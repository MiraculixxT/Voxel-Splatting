# Voxel Splatting
The idea behind this project is to implement a dynamic block game in 3D space,
but instead only using the usual vector rendering it should try to benefit from gaussian splatting.

Gaussian Splatting is a new technology to render photo realistic static scenes with low rendering cost.
Can this technology be used in game development to improve visuals and performance, or simplify aspects like transparency?

The challenges will be to adapt gaussian splatting into a dynamic game scene, which isnt based on a real object.
Also if useful on how to combine it with classic vector rendering.
